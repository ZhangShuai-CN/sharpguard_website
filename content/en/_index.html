SharpGuard is an extremely simple yet fast and modern open-source VPN platform.
Supports multiple VPN tunnels such as IPSec VPN, SSL/TLS VPN, Wireguard, etc.

Virtualizing security functions can run as VNFs on generic hardware. {{% /blocks/feature %}} {{% blocks/feature icon="fa-shuffle" title="High Performance" %}} A combination of extremely high-speed cryptographic primitives and use of kernel bypass technology means that secure networking can be very high-speed. {{% /blocks/feature %}} {{% blocks/feature icon="fa-key" title="Cryptographically Sound" %}} SharpGuard uses state-of-the-art cryptography, like the Noise protocol framework, Curve25519, ChaCha20, Poly1305, BLAKE2, SipHash24, HKDF, and secure trusted constructions. {{% /blocks/feature %}} {{% blocks/feature icon="fa-cubes" title="Composable Modules" %}} Modular design, flexible arch with many bonus features. {{% /blocks/feature %}} {{% blocks/feature icon="fa-shield-halved" title="Minimal Attack Surface" %}} SharpGuard has been designed with ease-of-implementation and simplicity in mind.
